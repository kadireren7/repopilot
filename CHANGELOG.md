# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-31

### Added
- Initial release of **RepoPilot**.
- **Repo Analyzer**: Detects languages, frameworks, and key infrastructure files.
- **Scoring Engine**: Calculates a "Ship Readiness Score" (0-100).
- **Stack-Aware Generator**: Generates `Dockerfile`, `.dockerignore`, and GitHub Actions for Next.js, Node, Python, and Go.
- **GitHub Integration**: Direct analysis of remote repos and automated Pull Request creation for fixes.
- **CLI UX**: Colored output, progress indicators (ora), and summary tables.
- **Configuration**: Support for `repopilot.config.json`.
- **Testing**: Basic test suite with Jest.
- **Open Source Assets**: MIT License, CONTRIBUTING.md, and Issue Templates.
